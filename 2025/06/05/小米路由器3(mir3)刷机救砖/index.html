小米路由器3(mir3)刷机救砖

小米路由器3刷入X-WRT

准备工作：

1.下载小米官方降级固件：https://mega.nz/file/tZ1TCKCJ#NtO9htNWrPzhh0UNA2dPtkSD3k67L9niIu2RRlawKfE

2.下载pb-boot: http://downloads.pangubox.com:6380/pb-boot/19.03.17/ Ctrl+f“xiaomi3” ，下载文件格式为img的

3.下载固件：http://opt.cn2qq.com/padavan/MI-R3_3.4.3.9-099.trx

4.使用小米WiFi app,获取官方的SSH：http://d.miwifi.com/rom/ssh

5.将官方的SSH文件与pb-boot一起放入U盘（FAT32格式）。

路由器降级：

1.登录小米路由器管理页面：192.168.31.1 在常用设置-系统状态，点击手动升级，将小米官方降级固件进行降级；

2.上传完成后，勾选清除当前所有用户配置

3.降级完成后，在192.168.31.1 重新配置路由器

开启小米路由器3的SSH：

1.请将下载的工具包bin文件复制到U盘(FAT/FAT32格式)的根目录下，保证文件名为miwifi ssh.bin;

2.断开小米路由器的电源，将U盘插入USB接口;

3.按住reset按钮之后重新接入电源，指示灯变为黄色闪烁状态即可松开reset键，等待3-5秒后安装完成

4.登录路由器管理页面，，取得网址栏stok值替换到此处，然后在浏览器中访问，逐条进行：
http://192.168.31.1/cgi-bin/luci/;stok=<你的stok值>/api/xqnetwork/set_wifi_ap?ssid=tianbao&encryption=NONE&enctype=NONE&channel=1%3Bnvram%20set%20ssh%5Fen%3D1%3B%20nvram%20commit

http://192.168.31.1/cgi-bin/luci/;stok=<你的stok值>/api/xqnetwork/set_wifi_ap?ssid=tianbao&encryption=NONE&enctype=NONE&channel=1%3Bsed%20%2Di%20%22%3Ax%3AN%3As%2Fif%20%5C%5B%2E%2A%5C%3B%20then%5Cn%2E%2Areturn%200%5Cn%2E%2Afi%2F%23tb%2F%3Bb%20x%22%20%2Fetc%2Finit.d%2Fdropbear

http://192.168.31.1/cgi-bin/luci/;stok=<你的stok值>/api/xqnetwork/set_wifi_ap?ssid=tianbao&encryption=NONE&enctype=NONE&channel=1%3B%2Fetc%2Finit.d%2Fdropbear%20start

SSH登录路由器，刷入pb-boot和固件：

1.U盘插入小米路由器3的USB口，使用SSH软件连接路由器，输入df -h 或lsblk查看是否识别U盘

2.查看mtd目录下的文件：cat /proc/mtd

3.切换到U盘：cd /extdisks/sda

4.将mtd目录的文件备份到U盘：
dd if=/dev/mtd0 of=mtd0.bin
dd if=/dev/mtd1 of=mtd1.bin
dd if=/dev/mtd2 of=mtd2.bin
dd if=/dev/mtd3 of=mtd3.bin
dd if=/dev/mtd4 of=mtd4.bin
dd if=/dev/mtd5 of=mtd5.bin
dd if=/dev/mtd6 of=mtd6.bin
dd if=/dev/mtd7 of=mtd7.bin
dd if=/dev/mtd8 of=mtd8.bin
dd if=/dev/mtd9 of=mtd9.bin
dd if=/dev/mtd10 of=mtd10.bin
dd if=/dev/mtd11 of=mtd11.bin
dd if=/dev/mtd12 of=mtd12.bin

5.逐行复制下面代码，刷入pb-boot：
nvram set uart_en=1
nvram set flag_last_success=1
nvram set boot_wait=on
nvram commit
mtd write pb-boot.img Bootloader #复制文件到路由器，出现最后一行“Writing from pb-boot.img to Bootloader …”表示成功！结束！

6.刷固件：把路由器跟电脑用网线连接路由器lan口，先不插入电源插头，按住恢复键3秒，再插入电源，再数5秒左右。此时在浏览器上输入192.168.1.1，进入管理界面。上传固件，点“恢复固件”；
这里大概率需要手动配置ip地址，如果输入192.168.1.1无法打开，手动配置后网卡后就可以打开pb-boot

救砖：

我在刷openwrt后路由器无法启动，使用路由器恢复模式重新刷机启动

必备：USB转TTL-CH340模块、杜邦线、排针 ：putty tftp 刷机固件（最好是潘多拉和老毛子的，不要用OPENWRT的。） CH340串口驱动程序

1.用网线把电脑连接路由器的lan口连接，路由器先不通电，连接USB转接ttl的线路：TTL端RX接路由器TX，TTL端TX接路由器RX，GND接GND，剩下一看情况这里需要注意，如何不接全是乱码就需要接上供电线3.3V

2.usb转ttl必要软件，不然无法通过ttl连接路由器写入命令。安装好之后在设备管理器里看一下端口号。

3.更改电脑ip，在网络和共享中心里面更改以太网ip为 192.168.1.3，子网掩码按tab键自动填充，网关和dns不填（默认网关填不填感觉没啥影响）

4.Tftp软件安装并启动，安装tftp后将固件文件 mi.trx拷贝到tftp的安装路径下，或者直接将 current directory 的文件路径替换为固件（mi.trx）所在路径，后面通过ttl命令，路由器经过Lan口的网线通过tftp来下载固件。

5.修改好ip后在tftp软件选项中选中server interfaces 为 192.168.1.3

6.然后把路由器的电源接好，不要通电，不要通电，不要通电。然后，打开putty软件，

7.选择你设备管理里边显示的端口号，刷新频率有115200、117200、117500，需要尝试，哪个显示正常滚屏信息就可以用。

8.通电后进入滚动信息阶段 按2键 ，然后会让你输入要刷入的文件名mi.trx，然后按y键

9.等待自动刷机就行。
