一、概述

RustScan 是用 Rust 编写的现代端口扫描器，核心卖点是极快——扫描全部 65535 个端口最快只需 3 秒。它自动将发现的开放端口管道传递给 Nmap 进行深度服务识别，形成了”快扫 + 深扫”的高效组合。

核心特点：

  • ⚡ 极速：全端口 3 秒扫描
  • 🔗 自动 Nmap 集成：发现端口后自动调用 Nmap
  • 📜 脚本引擎：支持 Python、Lua、Shell 自定义脚本
  • 🧠 自适应学习：根据环境自动优化参数
  • 🌐 IPv6 + CIDR 支持
  • ♿ 无障碍设计：首款注重无障碍的渗透测试工具

十、自适应学习

RustScan 会根据环境自动优化扫描参数。

自适应学习包括：

  • 自动调整 ulimit（文件描述符）
  • 根据网络状况自动调整批次大小
  • 记住上次扫描的优化参数

十三、Docker 用法

⚠️ Docker 注意事项：

  • 使用 --net=host 让容器使用宿主机网络（必须）
  • Docker 下可能需要调整 -b（批次大小）和 -t（超时）
  • 建议先在宿主机测试，确认参数后再用 Docker

维度 RustScan Masscan Nmap
语言 Rust C C
速度 ⚡ 极快（3秒全端口） 🚀 最快（10M+ pps） 🐢 较慢

最佳选择：

  • 🏃 快速发现：RustScan（最简单）或 Masscan（最快速）
  • 🔬 精细分析：Nmap
  • 🌍 大规模扫描：Masscan
  • 🎯 日常使用：RustScan（自动调用 Nmap，一站式）
