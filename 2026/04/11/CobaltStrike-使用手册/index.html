一、概述

Cobalt Strike（简称 CS）是一款商业化的红队渗透测试平台，集成了社工攻击、漏洞利用、后渗透、横向移动等完整的攻击链功能，支持团队协作和分布式操作。

核心特点：

  • 🎯 完整攻击链：社工 → 初始访问 → 权限提升 → 横向移动 → 数据窃取 → 持久化
  • 🔌 多协议 C2：HTTP / HTTPS / DNS / SMB / TCP Beacon
  • 🤖 Beacon：功能强大的后渗透 Agent
  • 👥 团队协作：多操作员同时操作同一目标
  • 🎭 流量伪装：Malleable C2 自定义 C2 流量特征
  • 📝 脚本扩展：Aggressor Script 自定义功能
  • 📊 报告生成：自动生成渗透测试报告
