一、概述

dirsearch 是一款基于 Python 开发的高级 Web 路径暴力破解工具，用于发现 Web 服务器上隐藏的目录和文件。相比 dirb 等经典工具，dirsearch 功能更丰富、过滤更精准、输出格式更多样。

核心特点：

  • 🎯 精准过滤：支持状态码、大小、文本、正则、重定向等多维度过滤
  • 📂 智能递归：普通递归 / 深度递归 / 强制递归，灵活控制
  • 📋 丰富输出：支持 JSON / XML / HTML / CSV / MD / SQLite / MySQL / PostgreSQL
  • 🔄 会话管理：支持中断后恢复扫描
  • 🔧 扩展名机制：%EXT% 占位符 + 强制扩展 + 覆盖扩展
  • 🎲 随机 UA：每次请求随机 User-Agent
  • 🌐 代理池：支持多代理轮换 + Tor
  • 📦 内置字典：自带分类字典（common / conf / vcs / backups / db / logs / keys / web）

目标设置

参数 说明
-u URL 目标 URL（可多次使用）
-l PATH URL 列表文件
--stdin 从 STDIN 读取 URL
--cidr=CIDR CIDR 网段目标
--raw=PATH 加载原始 HTTP 请求文件
--scheme=SCHEME 指定协议（http/https）
--nmap-report=PATH 从 nmap 报告加载目标

字典设置

参数 说明
-w WORDLISTS 自定义字典（逗号分隔多个）
--wordlist-categories=CATEGORIES 内置分类字典（common,conf,vcs,backups,db,logs,keys,web,all）
-e EXTENSIONS 扩展名（逗号分隔，如 php,asp）
-f 强制扩展：对每个词追加扩展名 + /
-O 覆盖扩展：替换已有扩展名
--exclude-extensions=EXT 排除扩展名
--remove-extensions 移除路径中的扩展名
--prefixes=PREFIX 添加前缀
--suffixes=SUFFIX 添加后缀（忽略目录）
-U 转大写
-L 转小写
-C 首字母大写

通用设置

参数 说明
-t THREADS 线程数
--async 异步模式
-r 递归扫描
--deep-recursive 深度递归（每层目录都递归）
--force-recursive 对所有发现路径递归（不仅目录）
-R DEPTH 最大递归深度
--recursion-status=CODES 递归的状态码条件
--subdirs=SUBDIRS 扫描指定子目录
--exclude-subdirs=SUBDIRS 排除子目录
--max-time=SECONDS 最大运行时间
--exit-on-error 遇到错误即退出

状态码与过滤

参数 说明
-i CODES 包含状态码（如 200,300-399）
-x CODES 排除状态码（如 404,500-599）
--exclude-sizes=SIZES 按大小排除（如 0B,4KB）
--exclude-text=TEXT 按文本内容排除
--exclude-regex=REGEX 按正则排除
--exclude-redirect=STRING 按重定向 URL 排除
--exclude-response=PATH 排除与指定路径响应相似的结果
--skip-on-status=CODES 遇到此状态码跳过目标
--min-response-size=LEN 最小响应长度
--max-response-size=LEN 最大响应长度

请求设置

参数 说明
-m METHOD HTTP 方法（默认 GET）
-d DATA POST 数据
--data-file=PATH 从文件加载 POST 数据
-H HEADERS 自定义 Header（可多次）
--headers-file=PATH 从文件加载 Header
-F 跟随重定向
--random-agent 随机 User-Agent
--auth=CREDENTIAL 认证凭据（user:pass 或 token）
--auth-type=TYPE 认证类型（basic / digest / bearer / ntlm / jwt）
--user-agent=UA 自定义 UA
--cookie=COOKIE Cookie
--cert-file=PATH 客户端证书
--key-file=PATH 客户端证书私钥

连接设置

参数 说明
--timeout=TIMEOUT 连接超时
--delay=DELAY 请求间延迟
-p PROXY 代理 URL（HTTP/SOCKS，可多次）
--proxies-file=PATH 代理列表文件（轮换使用）
--proxy-auth=CRED 代理认证
--replay-proxy=PROXY 仅将发现路径发代理
--tor 使用 Tor 网络
--max-rate=RATE 每秒最大请求数
--retries=N 失败重试次数
--ip=IP 指定服务器 IP
--interface=IFACE 指定网络接口

输出设置

参数 说明
-o PATH 输出文件路径
--format=FORMAT 输出格式（simple / plain / json / xml / md / csv / html / sqlite / mysql / postgresql）
--log=PATH 日志文件
-q 静默模式
--full-url 输出完整 URL
--redirects-history 显示重定向历史
--no-color 无颜色输出

高级设置

参数 说明
--crawl 从响应中爬取新路径

十三、会话管理

dirsearch 支持会话保存与恢复，中断后可继续扫描：

工作流程：

  1. 首次运行指定 -s session.json 保存会话
  2. 中断后运行 dirsearch -s session.json 恢复
  3. 使用 --list-sessions 查看所有可恢复会话

语言 Python Go C Go

上手难度 🟡 中 🔴 较高 🟢 最低 🟢 低
扩展名机制 ✅ %EXT% 占位符 ✅ -e ✅ -X ✅ -x
强制扩展 ✅ -f ❌ ❌ ❌

选择建议：

  • 🚀 快速入门 + 简单扫描：dirb（最简单）
  • 🎯 灵活 Fuzzing + 任意位置：ffuf
  • 📋 丰富输出 + 精准过滤 + 会话管理：dirsearch
  • ⚡ DNS 爆破 + 多协议：gobuster
  • 🔄 深度递归 + 分类字典：dirsearch
