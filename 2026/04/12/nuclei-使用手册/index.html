一、概述

nuclei 是 ProjectDiscovery 开发的基于 YAML 模板的高速漏洞扫描器。通过社区贡献的模板，可以快速检测 Web 应用、API、网络服务、DNS、云配置中的已知漏洞。

核心特点：

  • ⚡ 极快：并行扫描 + 请求聚类，速度远超传统扫描器
  • 🎯 零误报：模板模拟真实攻击步骤，减少误报
  • 📝 YAML 模板：简单易读，社区贡献了 10000+ 模板
  • 🔌 多协议：HTTP / DNS / TCP / SSL / WebSocket / WHOIS / JavaScript / Code / File
  • 🧠 DAST 模式：支持主动 Fuzzing
  • 🌐 OAST：集成 Interactsh 进行盲漏洞检测（SSRF / DNS 注入等）
  • 🔍 Uncover：集成 Shodan / Censys / FOFA / Hunter / ZoomEye 进行资产发现
  • 🖥 无头浏览器：支持 Headless 模板检测 XSS 等前端漏洞
  • 🔗 CI/CD 集成：可集成到 DevSecOps 流水线

4.2 参数总览

目标

参数 说明
-u, -target 目标 URL/主机（可多次）
-l, -list 目标列表文件
-eh, -exclude-hosts 排除的主机
-sa, -scan-all-ips 扫描 DNS 记录关联的所有 IP
-resume 恢复扫描
-im, -input-mode 输入模式（list / burp / jsonl / yaml / openapi / swagger）

模板选择

参数 说明
-t, -templates 指定模板/目录（逗号分隔）
-turl, -template-url 模板 URL
-w, -workflows 指定工作流
-nt, -new-templates 仅运行最新版本新增模板
-as, -automatic-scan 自动扫描（Wappalyzer 技术检测 → 标签映射）
-validate 验证模板语法
-tl 列出匹配当前筛选条件的模板
-tgl 列出所有可用标签
-code 启用 Code 协议模板
-esc, -enable-self-contained 启用 Self-contained 模板
-file 启用 File 模板

模板筛选

参数 说明
-tags 按标签筛选（逗号分隔）
-etags, -exclude-tags 排除标签
-itags, -include-tags 强制包含标签（即使被排除）
-id, -template-id 按模板 ID 筛选（支持通配符）
-eid, -exclude-id 排除模板 ID
-s, -severity 按严重级别（info / low / medium / high / critical）
-es, -exclude-severity 排除严重级别
-a, -author 按作者筛选
-pt, -type 按协议类型（dns / http / tcp / ssl / file / headless / websocket / whois / code / javascript）
-ept, -exclude-type 排除协议类型
-t, -templates 指定模板目录/文件
-et, -exclude-templates 排除模板目录/文件
-it, -include-templates 强制包含模板
-em, -exclude-matchers 排除特定匹配结果
-tc, -template-condition DSL 条件筛选

输出

参数 说明
-o, -output 输出文件
-j, -jsonl JSONL 格式输出
-silent 静默模式（仅显示发现结果）
-nc, -no-color 无颜色输出
-or, -omit-raw 省略请求/响应原始数据
-ot, -omit-template 省略模板内容
-nm, -no-meta 省略元数据
-ts, -timestamp 显示时间戳
-ms, -matcher-status 显示匹配失败状态
-sresp, -store-resp 保存所有请求/响应
-srd, -store-resp-dir 响应保存目录
-me, -markdown-export Markdown 格式导出
-se, -sarif-export SARIF 格式导出
-je, -json-export JSON 格式导出
-jle, -jsonl-export JSONL 格式导出
-rdb, -report-db 报告数据库
-rd, -redact 隐藏敏感字段

速率控制

参数 说明 默认值
-rl, -rate-limit 每秒最大请求数 150
-bs, -bulk-size 每模板并行主机数 25
-c, -concurrency 并行模板数 25
-pc, -payload-concurrency 每模板并行载荷数 25

请求配置

参数 说明
-fr, -follow-redirects 跟随重定向
-fhr, -follow-host-redirects 仅跟随同主机重定向
-mr, -max-redirects 最大重定向数（默认 10）
-dr, -disable-redirects 禁用重定向
-H, -header 自定义 Header（可多次）
-V, -var 自定义变量
-ev, -env-vars 启用环境变量
-sni 自定义 TLS SNI
-fh2, -force-http2 强制 HTTP/2
-ztls 使用 ztls 库
-tlsi, -tls-impersonate TLS JA3 随机化
-cc, -client-cert 客户端证书
-ck, -client-key 客户端密钥

优化

参数 说明 默认值
-timeout 超时秒数 10
-retries 重试次数 1
-spm, -stop-at-first-match 首个匹配即停止 关闭
-ss, -scan-strategy 扫描策略（auto / host-spray / template-spray） auto
-stream 流式模式 关闭
-mhe, -max-host-error 单主机最大错误数 30
-project 使用项目缓存避免重复请求 关闭
-nh, -no-httpx 禁用 httpx 探测非 URL 输入 关闭

DAST/Fuzzing

参数 说明
-dast 启用 DAST 模板
-dts, -dast-server DAST 服务模式（实时 Fuzzing）
-ft, -fuzzing-type Fuzzing 类型（replace / prefix / postfix / infix）
-fm, -fuzzing-mode Fuzzing 模式（multiple / single）
-fa, -fuzz-aggression Fuzzing 激进程度（low / medium / high）

Interactsh / OAST

参数 说明
-iserver 自建 Interactsh 服务器
-itoken Interactsh 认证 Token
-ni, -no-interactsh 禁用 Interactsh

Uncover

参数 说明
-uc, -uncover 启用 Uncover
-uq, -uncover-query 搜索查询
-ue, -uncover-engine 搜索引擎（shodan / censys / fofa / hunter / zoomeye 等）
-ul, -uncover-limit 结果数量限制（默认 100）

Headless

参数 说明
-headless 启用无头浏览器模板
-page-timeout 页面超时（默认 20s）
-sb, -show-browser 显示浏览器窗口
