一、概述

fscan 是一款用 Go 语言编写的内网综合扫描工具，集成了主机存活探测、端口扫描、服务爆破、漏洞检测、漏洞利用等功能，实现一键自动化、全方位内网扫描。

核心特点：

  • 🚀 一键扫描：fscan -h 192.168.1.0/24 全自动完成存活→端口→爆破→漏洞
  • 🔍 信息收集：ICMP 存活、端口扫描、NetBIOS 探测、域控识别
  • 🔑 密码爆破：SSH / SMB / RDP / FTP / MySQL / MSSQL / Redis / PostgreSQL / Oracle / MongoDB / Memcached
  • 🛡 漏洞检测：MS17-010、WebLogic、Struts2，兼容 XRay POC
  • 💥 漏洞利用：Redis 写公钥 / 写计划任务、SSH 命令执行、MS17-010 ShellCode
  • 🌐 Web 探测：网站标题、CMS 指纹、OA 框架识别
  • 📦 单文件：Go 编译，无依赖，跨平台

3.2 参数总览

目标设置

参数 说明 示例
-h 目标 IP / CIDR 192.168.1.1 / 192.168.1.0/24
-hf 从文件读取目标 targets.txt
-hn 扫描时跳过的 IP 192.168.1.1
-u 指定 URL 扫描 http://target.com
-uf 从文件读取 URL urls.txt

端口设置

参数 说明 默认值
-p 扫描端口 21,22,80,81,135,139,443,445,1433,3306,5432,6379,7001,8000,8080,8089,9000,9200,11211,27017
-pa 新增端口（追加到默认列表） 无
-pn 跳过的端口 无

扫描模式

参数 说明 默认值
-m 扫描模式 all

可用模式：
all / icmp / portscan / ssh / smb / rdp / ftp / mysql / mssql / redis / psql / oracle / mgo / mem / fcgi / netbios / findnet / web / webonly / ms17010 / cve20200796 / main

爆破设置

参数 说明
-user 指定用户名
-userf 用户名字典文件
-usera 在默认字典基础上追加用户名
-pwd 指定密码
-pwdf 密码字典文件
-pwda 在默认字典基础上追加密码
-domain SMB 爆破时设置域名
-sshkey SSH 私钥文件

扫描控制

参数 说明 默认值
-t 线程数 600
-time 端口扫描超时（秒） 3
-wt Web 访问超时（秒） 5
-np 跳过存活探测 关闭
-nobr 跳过密码爆破 关闭
-nopoc 跳过 Web POC 扫描 关闭
-num Web POC 发包速率 20
-ping 用 ping 代替 ICMP 探测 关闭
-debug 多久没响应打印进度（秒） 60

输出

参数 说明 默认值
-o 结果保存路径 result.txt
-no 不保存结果到文件 保存
-silent 静默模式 关闭

漏洞利用

参数 说明
-c SSH 命令执行
-rf Redis 写公钥文件（如 id_rsa.pub）
-rs Redis 计划任务反弹 Shell（ip:port）
-sc MS17-010 ShellCode（add / guest）
-path FCGI / SMB 远程文件路径

网络配置

参数 说明
-proxy HTTP 代理
-socks5 SOCKS5 代理
-cookie 设置 Cookie

Web 扫描

参数 说明
-pocname 指定 POC 名称（模糊匹配）
-pocpath 指定 POC 路径
