什么是 Wayback Machine

Wayback Machine 是由 互联网档案馆（Internet Archive） 提供的服务，存储网站随时间变化的快照。它充当网络的时间胶囊，保存着可能已经不再在线的数据。

  • 隐藏或遗忘的文件：访问不再可访问的旧配置或文件
  • 过时的安全措施：识别旧系统中的漏洞
  • 历史数据：分析网站的先前状态以获得更深入的洞察

访问（ 404 错误）已删除文件

步骤

  1. 前往 https://web.archive.org/
  2. 将你从 Wayback Archive 找到的 404 URL 粘贴到搜索栏中，然后按回车
  3. 你将看到该 URL 的完整时间线
  4. 从时间线中选择一个较旧的快照并点击它
  5. 这将为你提供已删除文件的下载链接

Robots.txt 文件

许多网站都有 robots.txt 文件，它为网络爬虫提供关于网站哪些部分可以访问、哪些应该避免的指令。有时这些文件包含有价值的数据，例如隐藏的端点或不打算公开的 URL。通过检查 Wayback Machine 中 robots.txt 文件的旧版本，你可以发现用于安全测试的额外端点。
