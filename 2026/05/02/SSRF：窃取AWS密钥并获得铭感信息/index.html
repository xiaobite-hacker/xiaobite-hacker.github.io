SSRF 测试清单

测试项 目标 示例
本地主机 内部服务 http://localhost:8080
云元数据 AWS/GCP/Azure 凭据 http://169.254.169.254
内网扫描 内部端口和服务 http://10.0.0.x:port
文件读取 本地文件 file:///etc/passwd
DNS 重绑定 绕过 IP 检查 evil.com → 127.0.0.1
URL 混淆 绕过过滤器 0xA9FEA9FE、十进制
Gopher 协议 内部服务利用 gopher://redis:6379
Webhook 服务器发起请求 callback_url 参数
PDF 生成 文件读取/XSS iframe、SVG 载荷

测试技巧

  1. 总是测试云元数据端点（169.254.169.254）
  2. 尝试各种 IP 表示（十六进制、十进制、IPv6）
  3. 测试文件协议（file:///）
  4. 检查是否有重定向跟随且不重新验证
  5. 寻找 webhook/回调功能——它们是 SSRF 的天然入口
