一、概述

Amass 是 OWASP 旗舰项目，用于攻击面映射和外部资产发现。它集成 80+ 数据源，支持被动枚举、主动探测、子域爆破、名称排列变换等多种技术。

核心特点：

  • 支持被动 / 主动 / 爆破三种模式
  • 80+ 数据源（API、搜索引擎、证书透明度、WHOIS、爬虫等）
  • 图数据库存储，支持增量扫描
  • 支持 PostgreSQL 远程数据库
  • 可视化关系图谱
  • 可配置的脚本引擎

三、子命令总览

子命令 用途
enum DNS 枚举与网络映射（核心）
intel 情报收集，发现更多关联域名
db 管理图数据库中的枚举结果

全局参数（所有子命令通用）：

参数 说明 示例
-config 指定 YAML 配置文件 amass enum -config config.yaml -d example.com
-dir 指定图数据库目录 amass enum -dir ./output -d example.com
-nocolor 禁用彩色输出 amass enum -nocolor -d example.com
-silent 静默模式 amass enum -silent -json out.json -d example.com
